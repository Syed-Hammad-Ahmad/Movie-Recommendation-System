# Movie Recommendation System

A content-based movie recommendation system built using Python, Machine Learning, NLP, and Streamlit.

## Features

- Recommends movies based on content similarity
- Processes 4,800+ movies from the TMDB dataset
- Uses CountVectorizer and Cosine Similarity
- Implements title normalization and fuzzy matching
- Displays movie posters using TMDB API
- Interactive Streamlit web application

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- NLTK
- Streamlit
- TMDB API

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
